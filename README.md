# Cursed Point Buy

Kleine Next.js Single-Page-App fuer ein Hybrid Point Buy System mit Curse-Regeln.

## Design

- Cleanes, modernes Single-Page-UI mit viel Weissraum und klarer Typografie.
- Dezente rote Akzente statt verspielter Fantasy-Optik.
- Externe Fonts: Google Fonts (`Space Grotesk`, `Manrope`).

## Start

```bash
npm install
npm run dev
```

Dann im Browser `http://localhost:3000` oeffnen.
